# LLM Quantization Thesis

**MTech Thesis — IIT Guwahati, 2026**

Investigating activation quantization schemes for efficient LLM inference. Extends SmoothQuant (Xiao et al., ICML 2023) by comparing per-tensor vs per-channel weight quantization under different smoothing strengths.

## Reference

- [SmoothQuant: Accurate and Efficient Post-Training Quantization for Large Language Models](https://arxiv.org/abs/2211.10438) — Xiao et al., ICML 2023
- [Official SmoothQuant repo](https://github.com/mit-han-lab/smoothquant)
