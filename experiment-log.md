# Experiment Log

Track EVERY run here — including failed ones. This is your research diary.

## Format

```
### [DATE] - [TASK] - [MODEL] - [SHORT DESCRIPTION]
- **Script:** path/to/script.py
- **Args:** --model X --scheme Y --alpha Z
- **GPU:** T4 / A100 (from Colab)
- **Duration:** ~Xm
- **Status:** ✅ Success / ❌ Failed / ⚠️ Partial
- **Result:** accuracy=XX.X%, perplexity=XX.X (or error message if failed)
- **Notes:** anything unusual, observations, next steps
- **Saved to:** Drive path or results/ path
```
